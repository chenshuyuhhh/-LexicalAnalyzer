### 使用说明
#### 1. 命令行执行
`LexicalAnalyzer.jar` 包放在本项目的out/artifacts/LexicalAnalyzer_jar目录下。
在 terminal 中执行如下语句
```shell
java -jar LexicalAnalyzer.jar file1 file2 ... filen
```
- file1、file2、filen 为文件路径参数，本项目将会对 file1、file2、filen 进行词法分析。
